Vuetify

Vuetify adalah Perpustakaan UI Vue dengan komponen Material yang Secara khusus setiap komponen dibangun dengan pendekatan mobile first untuk menjadi responsif dan sangat modular. Vuetify juga menyediakan template dasar untuk Apache Cordova, A La Carte, Electron, PWA, NUXT, Webpack, dan Simple HTML

Details

  • Dilengkapi dengan API yang mudah digunakan
  • Desain material memengaruhi perpustakaan
  • Fitur yang paling berguna meliputi: input formulir dan bereaksi untuk menggunakan input (mouseovers dan klik)

    Quasar

    Quasar adalah kerangka vue lain yang mengikuti Pedoman Material 2.0. Quasar memiliki hingga 81 komponen dapat memberikan dukungan untuk desktop dan mobile browser (termasuk iOS Safari!) di luar kotak. Dengan Quasar, Anda mendapatkan dukungan kelas satu untuk setiap mode build (SPA, SSR, PWA, aplikasi Seluler, aplikasi Desktop). Quasar adalah framework berbasis Vue.js open source yang memungkinkan pengembang untuk membangun aplikasi responsif untuk hampir setiap platform. Quasar memiliki kemampuan built-in untuk beralih di antara mode build.

    Buefy

    Buefy adalah open source, kerangka vue ringan yang dibangun di atas kerangka CSS Bulma.Bulma adalah framework CSS modern yang memungkinkan pengembang untuk secara intuitif membuat komponen responsif. Dengan nama-nama kelas bahasa Inggris yang sederhana, mudah untuk memahami sekilas bagaimana tepatnya nama kelas Anda mempengaruhi gaya elemen.

  • Tidak ada ketergantungan internal

  • Kinerja dan kegunaan yang andal
